## Running Tests

To run the Node.js application, use one of the following commands:

1. **Using `node .`**  
   Runs the Node.js app starting from the `index.js` file in the current directory.

   ```bash
   node .
   ```

2. **Using `node index.js`**  
   Directly runs the `index.js` file.

   ```bash
   node index.js
   ```
